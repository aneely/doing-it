doing it

Usage

Doing It is a non-app for keeping track of tasks. In the editor above, just type stuff and it styles it for you. Each line you type represents a task, a group of tasks, or a comment. It will saves your tasks too. (Go ahead! Close the window, come back, and your tasks will still be here, just as you left them)

Tasks

The first character of each line indicates the status of the task.
  • - an unfinished task
  • / a finished task
  • # a blocked task
  • ! a really important task

Groups

You can group tasks with a line that ends with a colon, like so:
Project A:
- Find a better project name
/ Figure out what to build
# tweet about it being done

Comments

Any line that isn't a task or a group is rendered as an italicized comment.

Caveats

  • Sorry, but Doing It will only work with Safari and Chrome
