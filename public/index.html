Doing It
Ahoy!
Here, each line represents a task, group, or comment.
Tasks
The first character of each line indicates the status of the task.
  • - an unfinished task
  • / a finished task
  • # a blocked task
  • ! a really important task
Groups
You can group tasks with a line that ends with a colon (:).
Comments
Any line that isn't a valid task or a valid group is rendered as an italicized comment.
