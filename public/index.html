doing it

Usage

Think of this page as a simple scratchpad that you can use to keep track of whatever it is you need to do. In the editor above, just type away and (if you follow the notes below), it will style things for you. Each line you type represents a task, a group of tasks, or a comment.

It also saves your tasks in your browser as you type. (Go ahead! Close the window, come back, and your tasks will still be here, just as you left them.)

Tasks

The first character of each line indicates the status of the task. The different task types are:
  • -an unfinished task
  • #a blocked task
  • !a really important task
  • /a finished task

Groups

You can group tasks with a line that ends with a colon, like so:
Project A:
- Find a better project name
/ Figure out what to build
# tweet about it being done

Comments

Any line that isn't a task or a group is rendered as an italicized comment.
